# TMDBTopRated

TMDBTopRated is an iOS application that displays the top-rated movies using The Movie Database (TMDB) API.

## Features

- List of top-rated movies
- Movie details

## Requirements

- iOS 15.0+
- Xcode 11.0+
- Swift 5.0+

## Installation

 Clone the repository
   
   ```sh
   git clone https://github.com/LgSus97/TMDBTopRaited.git
```
